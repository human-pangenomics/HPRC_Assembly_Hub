Description

This track represents the gene models for the assembly generated using the CAT (Compartive Annotation Toolkit) software. The reference annotations are from GENCODE V38 mapped with the Minigraph-Cactus T2T-CHM13-based genomic alignments.

Display Conventions and Configuration

This track follows the display conventions for gene prediction tracks. The exons for putative non-coding genes and untranslated regions are represented by relatively thin blocks, while those for coding open reading frames are thicker. Gene names are displayed in 'pack' or 'full' mode. More information about each gene can be found by clicking on the specific gene/transcript model.

The following color key is used:

  • Blue: protein coding
  • Green: non-coding
  • Pink: pseudogenes

Methods

Genome annotation for was performed using Comparative Annotation Toolkit (CAT). CAT leverages whole-genome alignments generated by Minigrahp-Cactus to transfer annotations from one source genome to one or more target genomes. For this annotation set, CAT lifted over the reference GENCODE V38 annotations this HPRC genome. The mapped transcripts were refined with AUGUSTUS to correct for genomic variation and assembly errors.
