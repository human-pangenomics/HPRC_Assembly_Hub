Description

RepeatMasker version 4.1.2-p1

Methods

Repeat masking on each assembly was performed iteratively by RepeatMasker. The first step masked using the default human repeat library, and the second step using a repeat library augmented by CHM13 satellite DNA sequences on the original assemblies after hardmasking the initial repeat masked DNA. The augmented repeat library is available at (https://zenodo.org/record/5537107#.YqNs13XMJrk), final_consensi_gap_nohsat_teucer.embl.txt, and a parallelized repeat masking pipeline is at (https://github.com/chaissonlab/segdupannotation), RepeatMaskGenome.snakefile. The union of the two steps yields the complete repeat masking
