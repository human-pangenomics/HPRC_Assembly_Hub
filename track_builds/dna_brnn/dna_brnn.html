Description

dna-brnn uses a Recurrent Neural Network (RNN) to quickly identify centromeric satellites (alpha satellite as well as human satellite I and II).

Methods

Source code for running the HPRC assemblies as a SnakeMake workflow is available on GitHub (https://github.com/mrvollger/Rhodonite/tree/v0.7-alpha).

References

Liao, Wen-Wei, Mobin Asri, Jana Ebler, Daniel Doerr, Marina Haukness, Glenn Hickey, Shuangjia Lu et al. "A draft human pangenome reference." bioRxiv (2022): 2022-07.

Li, Heng. "Identifying centromeric satellites with dna-brnn." Bioinformatics 35, no. 21 (2019): 4408-4410.
