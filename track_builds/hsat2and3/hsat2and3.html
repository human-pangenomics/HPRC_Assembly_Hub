Description

Human satellite 2 & 3 annotations created by running HSAT2/3 tooling from the T2T project CHM13. See GitHub for more information.

Methods

The HSAT2/3 identification tool takes a fasta file containing DNA sequences (HPRC assemblies in this case), and two provided text files containing HSat2 and HSat3 specific 24-mers (defined using HSat2/3 HuRef reads identified in Altemose et al. PLoS Comp Bio 2014) and outpus a BED file listing all contiguous regions in the reference likely to be HSat2 or HSat3, along with their strand orientation.
